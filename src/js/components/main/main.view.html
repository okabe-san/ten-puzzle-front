How to play?
There are 4 digits. You need to use all 4 digits and some operators to get "10". Calculation happens letf to right. Use "()" if you need them to change the calculation order. For example;
2 + 2 * 2 + 2 = 10
Because 2 + 2 = 4, 4 * 2 = 8, 8 + 2 = 10
If use (), it will be 2 + (2 * 2) + 2 = 8
Because 2 * 2 = 4, 2 + 4 = 6, 6 + 2 = 8

Buttons
"Next" - Change 4 digits.
"Try again" - Delete calculation, but 4 digits are same.
"Submit" - Submit the answer.
"Record Book" - To check a game record.
